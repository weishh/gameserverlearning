## 1️⃣ 告诉我你要继续学习

在对话开始时说：
```
"我要继续学习游戏服务器开发，请查看 PROGRESS.json 恢复我的进度"
```

## 2️⃣ 我会自动加载你的进度

我会读取以下文件：
- **PROGRESS.json** - 你的学习进度和状态
- **LEARNING_LOG.md** - 你的学习日志
- **最近的代码文件** - 检查你的实现

## 3️⃣ 继续你的学习计划

根据你的进度，我会：
- 提醒你上次学到哪里
- 建议今天的学习内容
- 帮你解决遗留的问题
- 继续下一个模块
